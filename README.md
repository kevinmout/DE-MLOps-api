# DE-MLOps-api

## Commands

Install FastAPI and uvicorn:

```
pip install uvicorn
```

```
pip install fastapi
```

Set environment variables:

1. Please copy contents of .env.example.
2. Create a new file called .env
3. Check the url of your website.
4. Replace this in .env.

To run API:

```
uvicorn main:app --port 4000
```
